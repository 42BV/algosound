# AlgoSound

Companion source code for the NL-JUG Java Magazine #3 2020 article 'Fun met Functies: Geluid'.

## Build

```
mvn clean install
```

## Run

Turn down the volume to protect the innocent.

```
java -jar ./target/algosound-0.0.1-SNAPSHOT.jar
```
